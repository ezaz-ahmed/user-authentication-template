## Description

This is a kickstart template. Main goal of this template is to start a backend server with [mongoose](https://mongoosejs.com/) & [express.js](https://expressjs.com/) without spending too much time on server.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
